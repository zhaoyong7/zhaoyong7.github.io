yii1－源码：错误信息处理机制

发表于 2015-08-04   |   分类于 程序员世界   |  

yii1的错误异常处理机制是通过phpset_error_handler;set_exception_handler内置函数处理的。

CApplication构造函数初始化了yii的系统处理函数initSystemHandlers

/**
 * Initializes the class autoloader and error handlers.
 */
protected function initSystemHandlers()
{
    if(YII_ENABLE_EXCEPTION_HANDLER)
        set_exception_handler(array($this,'handleException'));
    if(YII_ENABLE_ERROR_HANDLER)
        set_error_handler(array($this,'handleError'),error_reporting());
}

yii将错误处理分配给handleError;异常处理机制分配给handleException。

  • UIDynamicItem 就是物理体系可以影响到的物体，像现实世界的桌子、椅子。在ios我们常见的UI控件像UIButton、UIView等实现了该接口。
