没有curl,如何进行服务器端请求？

发表于 2015-06-25   |   分类于 程序员世界   |  

stream_contex_create(array)

不依赖curl，php已经有了相关的函数实现stream_contex_create

实现方法很简单：

$data = array(
    'foo'=>'bar', 
    'baz'=>'boom', 
    'site'=>'www.baidu.com', 
    'name'=>'nowa magic'
); 
$data = http_build_query($data); 
$options = array(
    'http' => array(
        'method' => 'POST',
        'header' => 'Content-type:application/x-www-form-urlencoded',
        'content' => $data
        'timeout' => 60 * 60 // 超时时间（单位:s）
    )
);
$url = "www.baidu.com";
$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);
echo $result;
