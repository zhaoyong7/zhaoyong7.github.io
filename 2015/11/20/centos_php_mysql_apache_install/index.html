CentOS下源码安装php、apache、mysql

发表于 2015-11-20   |   分类于 程序员世界   |  

一、准备安装包：wget “xxxxxx” :

  1. PHP: (wget –content-disposition “url”:获取安装包)

    http://cn2.php.net/get/php-5.6.4.tar.gz/from/this/mirror

  2. Mysql:

    http://dev.mysql.com/get/Downloads/MySQL-5.5/mysql-5.6.16.tar.gz

  3. Apache(需要依赖apr、apr-util和pcre):

    • apr

      http://apache.fayea.com//apr/apr-1.5.2.tar.gz

    • apr-util

      http://apache.fayea.com//apr/apr-util-1.5.4.tar.gz

    • pcre

      ftp://ftp.csx.cam.ac.uk/pub/software/programming/pcre/pcre-8.37.tar.bz2

    • apache

      https://archive.apache.org/dist/httpd/httpd-2.4.17.tar.bz2

二、安装：

  1. 安装Apache：

    • 安装apr依赖：

      cd apr-1.5.2

      ./configure –prefix=/usr/local/apr

      make && make install

    • 安装apr-util依赖：

      cd apr-util-1.5.4

      ./configure –prefix=/usr/local/apr-util –with-apr=/usr/local/apr

      make && make install

    • 安装pcre依赖：

      cd pcre-8.37

      ./configure

      make && make install

    • 安装apache:

      cd httpd-2.4.17

      ./configure –prefix=/usr/local/apache2 –with-apr=/usr/local/apr –with-apr-util=/usr/local/apr-util –with-pcre=/usr/local/pcre –enable-so –enable-rewrite

      make && make install

  2. 安装Mysql:

    • 安装cmake：

      yum -y install cmake make

    • 设置mysql用户和组：

      groupadd mysql

      useradd -r -g mysql mysql

    • 安装mysql：

      cd mysql-5.6.16

      cmake -DCMAKE_INSTALL_PREFIX=/usr/local/mysql -DMYSQL_DATADIR=/usr/local/mysql/data -DSYSCONFIGDIR=/usr/local/mysql/etc -DWITH_INNOBASE_STORAGE_ENGINE=1 -DWITH_ARCHIVE_STORAGE_ENGINE=1 -DWITH_BLACKHOLE_STORAGE_ENGINE=1 -DWITH_PARTITION_STORAGE_ENGINE=1 -DMYSQL_UNIX_ADDR=/tmp/mysqld.sock -DMYSQL_TCP_PORT=3306 -DENABLED_LOCAL_INFILE=1 -DEXTRA_CHARSETS=all -DDEFAULT_CHARSET=utf8 -DDEFAULT_COLLATION=utf8_general_ci -DMYSQL_USER=mysql

      make && make install

      注：编译错误后修改完毕需要删除CMakeCache.txt文件，再重新编译。

    • 修改mysql目录所有者和组：

      cd /usr/local/mysql

      chown -R mysql:mysql .

    • 初始化mysql数据库：

      cd /usr/local/mysql/scripts

      ./mysql_install_db –user=mysql –datadir=/usr/local/mysql/data

    • 复制mysql服务配置文件：

      cp /usr/local/mysql/support-files/my-default.cnf /etc/my.cnf

      注：存在则覆盖。

    • 复制mysql启动脚本至开机启动服务目录（选）：

      cp /usr/local/mysql/support-files/mysql.server /etc/init.d/mysqld

      注：此时可以利用service mysqld start | stop | restart….，启动mysql脚本

    • 加入开机启动项目（选）：

      chkconfig –level 35 mysqld on

    • mysql加入加入PATH路径（选）：

      vim /etc/profile

      PATH=/usr/local/mysql/bin:/usr/local/mysql/lib:$PATH

      source /etc/profile

    • 修改mysql用户root密码（选）：

      ./usr/local/mysql/bin/mysqladmin -u root password ‘123456’

      注:登陆mysql客户端时候，mysql -u root -p，此时需要输入密码，没设置则不需要

  3. 安装PHP：

    • 安装：

      cd php-5.6.4

      ./configure –prefix=/usr/local/php –with-apxs2=/usr/local/apache2/bin/apxs –with-libxml-dir=/usr/include/libxml2 –with-config-file-path=/usr/local/apache2/conf –with-mysql=/usr/local/mysql –with-mysqli=/usr/local/mysql/bin/mysql_config –with-gd –enable-gd-native-ttf –with-zlib –with-mcrypt –with-pdo-mysql=/usr/local/mysql –enable-shmop –enable-soap –enable-sockets –enable-wddx –enable-zip –with-xmlrpc –enable-fpm –enable-mbstring –with-zlib-dir –with-bz2 –with-curl –enable-exif –enable-ftp –with-jpeg-dir=/usr/lib –with-png-dir=/usr/lib –with-freetype-dir=/usr/lib/

      注：遇到reinstall依赖，直接yum安装即可，当遇到Cannot allocate memory时，在编译的时候加上–disable-fileinfo，重新编译。

      make && make install

  4. apache配置httpd.conf修改支持php(额外)：

    vim /usr/local/apache2/conf/httpd.conf

    • 添加php支持

      AddType application/x-httpd-php .php .phtml

      AddType application/x-httpd-php-source .phps

    • 添加默认索引页面index.php

      找到“DirectoryIndex”，在index.html后面加上”index.php”

      DirectoryIndex index.html index.php

    • 开启Apache支持伪静态

      找到“AllowOverride None”，修改为AllowOverride All
