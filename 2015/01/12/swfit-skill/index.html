swfit编程技巧总结

发表于 2015-01-12   |  

###1.属性监视器的巧妙运用：
在做ios开发的时候，我们经常有这样的需求，当一个属性，更改时，相对应的UI控件表现出来的状态也要改变，最明显的就是，当我们做一个计时器，显示分钟和秒数需要格式化，传统我们是在把实际秒数和视图表现写在回调函数中。这样当我们有多个回调函数时，代码就很冗余了。

我们可以用更swift的方式来编写我们的代码。

var changeSecond : Int = 0 {
     willSet(newSecond){
         let min = newSecond / 60
         let second = newSecond % 60 
         labbel!.text = NSString(format:"%02d:%02d",min,second)
     }    
 }

###2.空合运算符:??

空合运算符(a ?? b)将对可选类型a进行空判断，如果a包含一个值就进行解封，否则就返回一个默认值b.这个运算符有两个条件:

  • 表达式a必须是Optional类型
  • 默认值b的类型必须要和a存储值的类型保持一致

相当于a != nil ? a! : b

var value = min(max(someValue,0),100)
